# cgpa-calculator
Streamlit GPA/CGPA calculator with Dark Blue Theme+Times New Roman Font


A beautiful Streamlit app to calculate GPA/CGPA for university students in Pakistan.

### **Features:**
- 🌙 Dark Blue Theme 
- 🔤 Times New Roman Font
- 📊 Full grading: A+ 4.0 → F 0.0
- ⚡ Real-time CGPA calculation

### **How to Run:**
```bash
pip install streamlit pandas
streamlit run cgpa.py
